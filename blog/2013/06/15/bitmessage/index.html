Bitmessage

Jun 15th, 2013

Bitmessage 是一種 P2P 的通訊協定，可以讓你發訊息給另外一個人，或是任意數量的訂閱者。

安裝在 Max OS X 上

我自己嘗試安裝 Bitmessage 在 Max OS X 10.8 上，很可惜目前只有 Windows 版的程式可直接下載， 還好官方網站提供了由源碼安裝的方法， 請見說明。 其中安裝 pyqt 時如果遇到錯誤

ImportError: No module named sipconfig

那表示你還需要安裝 SIP ，請到此處下載。 解壓縮檔案後到目錄下使用以下指令安裝：

python configure.py
make
sudo make install

現在繼續執行 brew install pyqt 就沒問題了。
